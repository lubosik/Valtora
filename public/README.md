# Public Assets

Place the following assets in this directory:

- `valtora-logo.png` - Primary Valtora logo (recommended: 360x120px or similar aspect ratio)
- `valtora-favicon.png` - Favicon (recommended: 32x32px or 64x64px)

The logo component (`src/components/ValtoraLogo.tsx`) is structured to easily swap between PNG, SVG, or inline SVG implementations.
